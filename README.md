# Instituto Raiz da Esperança — Entrega I/II/III Integradas

Projeto final integrando as três etapas da disciplina:
- Entrega I: Estrutura HTML semântica (index, projetos, cadastro).
- Entrega II: Sistema de design, CSS modular e componentes.
- Entrega III: SPA (Single Page Application), templates JavaScript, validação de formulários e salvamento em localStorage.

## Como testar localmente
1. Extraia o ZIP.
2. Abra `index.html` no navegador.
3. Navegue entre Início / Projetos / Doação / Sobre / Contato (conteúdo carregado via SPA sem reload).
4. Em Doação, faça um envio inválido pra ver as mensagens; envie corretamente pra ver o toast de sucesso (simulado).
